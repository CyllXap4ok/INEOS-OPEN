Есть общие правила для графиков (они у большинства издательств):
1. Без фона
2. Без сетки внутри шкал (если мешает восприятию)
3. Соразмерность шрифтов
4. Только сильные (темные) цвета, т.е. никаких желтых, циан, салатовых, розовых.
5. Легенда для кривых.
